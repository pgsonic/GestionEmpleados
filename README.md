JORNADAS, INGRESO Y EGRESO:
. Esta jornada, dependiendo del CONVENIO al que pertenezca el trabajador, tendrá una HORA DE INICIO y HORA DE FINALIZACIÓN preestablecida para cada dia de la semana.
